#  CNN - Classification MNIST & Détection Cancer du Sein

Projet de Mathématiques pour le Machine Learning — EFREI Paris 2025-2026.

##  Objectif

Implémenter des réseaux de neurones de A à Z (sans boîtes noires) pour résoudre
des problèmes de classification d'images, depuis les chiffres manuscrits jusqu'à
la détection de cancers du sein.

##  Structure du projet

| Partie | Dataset | Statut |
|--------|---------|--------|
| Partie 1 - MNIST | Chiffres manuscrits 28×28 | ✅ Disponible |
| Partie 2 - CIFAR-10 | Images couleur 32×32 | 🔜 En cours |
| Partie 3 - Mammographies | CBIS-DDSM | 🔜 En cours |

##  Partie 1 : MNIST

- Modèle linéaire multi-classe (softmax + cross-entropy)
- Réseau de neurones multicouches (H=1 et H=2 couches cachées)
- Rétropropagation implémentée manuellement
- Analyse des erreurs et visualisation des chiffres mal classés
